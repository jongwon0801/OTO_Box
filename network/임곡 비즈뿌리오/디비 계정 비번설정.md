#### 비즈뿌리오 uds.conf 새로 할떄 예시
```less
DBNAME = MYSQL
DBURL = jdbc:mysql://localhost:3306/yellowbox?useUnicode=true&characterEncoding=utf-8&useSSL=false&allowPublicKeyRetrieval=true
DBUSER = yellowbox
DBPASS = tmshdnxmfl
```

#### yellowbox 계정으로 디비 접속
```less
mysql -u yellowbox -p
```

#### yellowbox 디비 공개 접속 계정 비번 변경
```less
ALTER USER 'yellowbox'@'localhost' IDENTIFIED BY 'tmshdnxmfl';
FLUSH PRIVILEGES;

SELECT User, Host FROM mysql.user; 
```
